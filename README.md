# Siap Polisi App

Aplikasi Polisi Panic Button, dibuat menggunakan Flutter versi 2.5.3

<br>

## Fitur

- Splashscreen
- Halaman Login dan Daftar
- Halaman Verifikasi
- Halaman Home Panic Button
- Halaman Detail Panic Button
- Halaman Unlock Laporan Masyarakat
